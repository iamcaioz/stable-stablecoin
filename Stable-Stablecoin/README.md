# Stable Stablecoin

Stable é uma stablecoin 100% colateralizada em multiplas stablecoins. Ao invés de somente um âncora, esse projetos acompanha o valor médio de diversas stablecoins.

## Tecnologias

- Solidity
- Foundry 
- Slither(For tests)

## Pré requisitos

- foundry

## Instalação

````bash

git clone https://github.com/iamcaioz/stable-stablecoin
cd Stable-Stablecoin
forge install
forge build

````

### Coverage
 
```bash

forge coverage

```
